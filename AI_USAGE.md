# AI_USAGE.md

## 1. AI Tools Used

* ChatGPT

---

## 2. Prompts Used

Examples of prompts I used during development:

* Review my implementation and suggest improvements.
* Review my React and Node.js project structure.
* Suggest improvements for API design and validation.
* Suggest UI/UX improvements for the stepper form experience.
* Help identify deployment and configuration issues.
* Review code quality and suggest optimizations.

---

## 3. What I Modified From AI Output

AI suggestions were used as guidance and were modified based on project requirements.

Changes made by me include:

* Designed the overall application architecture
* Designed the database structure for forms and submissions
* Designed API endpoints and request/response flow
* Implemented dynamic form rendering logic
* Implemented draft save and resume functionality
* Implemented progress tracking
* Customized frontend components and UI behavior
* Modified validation logic to match the project schema
* Adjusted deployment configuration for the final setup

The final implementation decisions were made by me.

---

## 4. What AI Got Wrong

Some AI suggestions required corrections after testing:

* A few suggestions did not fully align with the assignment requirements and needed to be adjusted.
*  In some cases, AI repeatedly suggested approaches that were not suitable for the project's structure, leading to repeated     troubleshooting before identifying the correct solution.
* Initial deployment recommendations required project-specific changes
* CORS configuration needed adjustments after deployment
* Some validation examples did not fully match the application's schema
* Environment variable setup required modification based on hosting platforms
* Some generated code examples needed refactoring to fit the project architecture

---

## 5. How I Verified Correctness

I verified the application through manual testing and review.

Verification included:

* Creating new forms and submissions
* Saving drafts and resuming them later
* Completing forms successfully
* Testing required field validations
* Testing invalid inputs and edge cases
* Verifying MongoDB data persistence
* Testing backend API responses
* Testing frontend and backend deployments
* Verifying CORS and rate limiting behavior
* Reviewing application flow end-to-end

---

## Additional Notes

AI was primarily used for suggestions, troubleshooting, and improvement ideas.

The application's architecture, API design, database modeling, business logic, form flow, deployment setup, and core functionality were designed and implemented by me. AI assisted as a development aid, while implementation decisions and final code integration remained my responsibility.
